# Jazz Hong Portfolio

Personal portfolio website for Jazz Hong, built with Hugo and deployed to GitHub Pages.

Live site: <https://jazz-hong.github.io/>

## Structure

- `content/` - site pages and page metadata
- `layouts/` - shared Hugo templates
- `static/` - CSS, JavaScript, images, PDFs, and standalone demos
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow

## Local Preview

Install Hugo Extended, then run:

```bash
hugo server
```

The production build is generated by GitHub Actions whenever `main` is pushed.
